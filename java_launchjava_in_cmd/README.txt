Pour lancer un fichier Java en ligne de commande:

Ecrire un fichier Java. ATTENTION => Le nom du fichier doit être le même que le nom de la classe que l'on écrit.

Ex: 
Hello.java et nom de classe Hello -> compilera
hello.java et nom de classe Hello -> ne compilera pas ! Le "h" de hello.java n'est pas en majuscule !

Pour compiler le fichier java, il faut lancer la commande suivante:
javac NomDuFichier.java -> cette commande créée un fichier NomDuFichier.class

Ensuite, pour exécuter le fichier, il suffit de lancer la commande suivante:
java NomDuFichier
